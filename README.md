# COM206--Midterm-Project
###
Midterm Project submission by Fidan Alasgarli 
###
Research Question / Problem Statement
###
How can users efficiently manage their daily tasks using a simple and user-friendly mobile application?
###
Many people struggle to keep track of tasks, deadlines, and priorities.
###
A structured digital solution is needed to organize tasks with names and descriptions in an accessible way.
###
Objective of the Project
###
The project aims to design and develop a mobile to-do list manager using Flutter and Dart. 
###
The application will allow users to add tasks with a name and description, view their tasks, and navigate easily using a menu-based interface.
